Counting occurrences

Implement an algorithm that counts how many times the target element occurs in a list of numbers. Try to implement the linear search algorithm yourself instead of using the built-in methods available in Python!

Input: the first line contains the elements of a list separated by spaces. The second line contains the target element.

Output: the number of occurrences of the target element in the list. If the element doesn't occur in the list, the algorithm should return 0.

Don't forget to process the input data and print the result of the algorithm!

Instead of the index, start by creating a variable for the count with the initial value 0.


Sample Input:
1 3 5 1 2 4
1

Sample Output:
2



Memory limit: 256 MB
Time limit: 15 seconds

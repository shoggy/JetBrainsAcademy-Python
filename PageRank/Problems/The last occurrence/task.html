The last occurrence

In some cases, it might be necessary to find the index of the last occurrence of some element in the list. Your task here is to modify the described version of linear search so that it works in this way.

Input: the first line contains the elements of the input list separated by space. The second line contains the target element.

Output: the index of the last occurrence of the target element in the list or $-1$ if the target element is not contained in the list.

Don't forget to process the input data and print the result of the algorithm!


Sample Input:
3 5 9 11 9
9

Sample Output:
4

The first occurrence in a range

Implement an algorithm that finds the first occurrence of a target element in a sublist of integer numbers.

Input: the first line contains the elements of a list separated by spaces. The second line contains the target element. The next line contains two numbers $a$ and $b$ ($a < b$) separated by space.

Output: the index of the first occurrence of the target element in the sublist starting with the index $a$ (inclusive) and ending with the index $b$ (exclusive) or $-1$ if the target element is not contained in the specified sublist.

Don't forget to process the input data and print the result of the algorithm!


Sample Input:
5 3 1 2 4
1
1 4

Sample Output:
2



Memory limit: 256 MB
Time limit: 15 seconds

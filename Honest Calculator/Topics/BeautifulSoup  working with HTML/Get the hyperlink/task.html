Get the hyperlink

Given the link to a web page, write a program that prints a hyperlink to the specific act of "Pygmalion" by Bernard Shaw. There are two lines in the input: the number of the act (be careful: the indexes start from 0) and the link to a web page with the text of "Pygmalion". For example, the input can look like this:

3
http://www.gutenberg.org/files/3825/3825-h/3825-h.htm

There are several hyperlinks on this page, which can be used to jump to one of the play's five acts.

In response to the input above, your program would need to print the following hyperlink: #act2. Note that you shouldn't print the WHOLE page link, just this small part.

Your algorithm to the task may look as follows. First, find all anchors. Then, choose the act from the list. Finally, get the href attribute using the get method.


Memory limit: 256 MB
Time limit: 30 seconds

Comparison of two arrays

There are four elements on each line in the input. Use them to create two arrays: the first two elements are used for the first array, the next two elements are for the second one. So [6, 7] and [9, 0]. Then check whether the elements of the first array are less than the elements of the second one. Print the resulting Boolean array.


Sample Input:
6
7
9
0

Sample Output:
[ True False]



Memory limit: 256 MB
Time limit: 15 seconds

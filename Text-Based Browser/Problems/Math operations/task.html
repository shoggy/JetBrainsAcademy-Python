Consider the following decorator: it takes a function with two arguments and prints the arguments before calling the function.

def print_info(func):
    def wrapper(arg1, arg2):
        print("The arguments of the function are:", arg1, arg2)
        return func(arg1, arg2)

    return wrapper

Let's say we use this decorator for a function that simply returns the sum of the two arguments. Then, the result for the values 22 and 25 will be the following:
